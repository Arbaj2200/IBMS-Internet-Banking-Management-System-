# 💳 Internet Banking Management System (IBMS)

## 📌 Overview
Internet Banking Management System (IBMS) is a web-based application developed using Java and Spring Boot. It simulates core banking functionalities such as user registration, login, fund transfer, and transaction management with secure authentication.

---

## 🚀 Features
- User Registration & Login
- Secure Password Encryption using BCrypt
- Fund Transfer (UPI / Account-based)
- Transaction History
- Account Management
- Input Validation & Error Handling

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot, Hibernate (JPA)
- **Frontend:** JSP, HTML, CSS
- **Database:** MySQL / PostgreSQL
- **Security:** BCrypt Password Hashing
- **Architecture:** MVC Pattern, REST API
